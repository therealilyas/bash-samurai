# 01-Terminal-Basics – Terminal va Fayl Tizimi

## Asosiy buyruqlar
| Buyruq | Tavsif |
|--------|--------|
| `ls` | Papka ichidagi fayllarni ko‘rsatish |
| `cd` | Papkani o‘zgartirish |
| `mkdir` | Yangi papka yaratish |
| `touch` | Yangi fayl yaratish |
| `pwd` | Hozirgi ishchi katalogni ko‘rsatish |
