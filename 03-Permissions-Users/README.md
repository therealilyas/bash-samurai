
# 3. Fayllar bilan ishlash

**cat, less, head, tail**

```bash
cat file.txt
less file.txt
head -n 10 file.txt
tail -n 5 file.txt
```

**cp, mv, rm**

```bash
cp source dest
mv oldname newname
rm file.txt
rm -r folder
```

**Amaliy mashq 3**

* Vazifa: `notes.txt` yarating, unga 5 qatordan iborat matn yozing, `head` va `tail` bilan 2 qatordan oling.
* Kutilgan output (`head -n 2 notes.txt`): birinchi 2 qator matn.
